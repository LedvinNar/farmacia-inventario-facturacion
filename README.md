## 📝 Descripción del Sistema

Este sistema web permite la gestión integral de inventario, ventas y facturación de una farmacia, facilitando el control de productos, clientes, movimientos de stock y generación de reportes.

El sistema ha sido desarrollado utilizando el framework Laravel bajo arquitectura MVC y una base de datos relacional MySQL.

---

## 🚀 Funcionalidades Principales

- ✔️ Gestión de Clientes (CRUD)
- ✔️ Gestión de Productos (CRUD)
- ✔️ Registro de Ventas
- ✔️ Facturación automática
- ✔️ Control de Stock en tiempo real
- ✔️ Reportes de ventas
- ✔️ Dashboard con gráficos
- ✔️ Sistema de Roles y Permisos:
  - Administrador
  - Vendedor
  - Consulta
- ✔️ Respaldo y Restauración de Base de Datos
- ✔️ Seguridad mediante cifrado de contraseñas
- ✔️ API para comunicación Frontend - Backend
